# Changelog

## 1.0.1

- Added `FFT.MoreDecraftingRecipes` equip-based recipe unlock patch for supported arrow decrafting recipes.
- Reworked `FFT.Knives_Master` unlock logic to strict paired dagger/knife conversion mapping.
- Added standalone `FFT.Beard_Additions` patch with internal `TextureOverrides/stormcancer-Beard_Additions` source layout.
- Added auto-sync of `manifest.json` dependencies from installed profile plugins during build.
- Expanded package dependencies to match installed profile mod requirements.

## 1.0.0

- First public release of `FFTweaks`.
- Added `FFT.TrueHardcore` compatibility patch for loot interaction behavior.
- Added `FFT.KnivesMaster` recipe unlock support for dagger recipes.
